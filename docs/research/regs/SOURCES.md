## Главные пробелы, требующие доп. работы (по убыванию важности для мониторинга ОСМС/ГОБМП)

1. **Приложения (формы, тарифы, коэффициенты) отсутствуют почти везде** — ни в `pravila_zakupa.*`, ни в `pravila_oplaty.*` не получены приложения (заявки, протоколы, тарифные таблицы, коэффициенты затратоемкости, перечни МКБ-10). Для хакатон-агента, которому нужны конкретные формы/тарифы, это самый крупный пробел.
2. `pravila_oplaty.ru.txt` / `.kk.txt` обрываются до глав об оплате конкретных профилей помощи (онкология, туберкулез, ВИЧ, психиатрия, село) — если эти профили релевантны Task 07, потребуется отдельный дозапрос.
3. `pravila_zakupa.kk.txt` заметно менее полон, чем русская версия того же акта (~30-35% против ~55-65%) — при двуязычном сравнении текста закупа опираться в основном на `pravila_zakupa.ru.txt`.
4. Источник URL не сохранён для `zakon_206_viii.ru.txt` / `.kk.txt` — стоит доустановить при следующей сверке.
5. Повторный фетч `pravila_zakupa.ru.txt` (второй раз, 2026-07-08) не продвинулся дальше того же места обрыва — похоже на ограничение объёма ответа инструмента WebFetch на этой конкретной странице, а не на случайную сетевую ошибку. Дальнейшие попытки тем же способом вряд ли дадут больше текста; если понадобится остаток документа, вероятно, потребуется постраничная навигация по оглавлению документа на самом сайте (открывать конкретные якоря/разделы по отдельности) вместо запроса всей страницы целиком.
