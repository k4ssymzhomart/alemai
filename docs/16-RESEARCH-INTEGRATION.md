## 4. Impact math v2 (gp14-real scale, honest)

Per clinic (31k, ~1,2 млрд ₸/год): снятия 1,5–3% → firewall saves ~10–20 млн ₸; недовыставление 0,5–1% → 6–12 млн; возражения-таймеры (перестали пропускать окна) → 3–8 млн; корректировка по пп. 25/26 вовремя → 10–20 млн. **Итого ~30–60 млн ₸/год ≈ 2,5–5% договора** + ~1 FTE времени. Астана (14 ГП разного масштаба, скейл по прикреплённым ≈ 8–10× ГП№14 суммарно) → **0,3–0,6 млрд ₸/год**. Национально: 0,5% от 3,4 трлн = **17 млрд ₸/год**. Все цифры — модельные, помечать; бенчмарк реальности: Фонд сам оспаривает 38 млрд/год — наш порядок величин консервативен.
